## Soʻz boshi

Bu qoʻllanmaning terminal versiyasi. Bu yerdagi barcha Git buyruqlari terminal orqali bajariladi. Agar GFIʼni afzal koʻrsangiz, [GitHub Desktopʼdan](/translations/README/UZBEK.md) foydalaning.

### Bu kim uchun?

Terminal orqali sodda Git buyruqlarini ishga tushura oladigan barcha. Mutaxassis boʻlishingiz talab etilmaydi — agar buyruqlardan foydalanib, katalogda ishlay olsangiz bas.

### Qanday hissa qoʻshishingiz mumkin?


Siz manashunday karta toʻldirish orqali [loyiha web sahifasiga](https://syknapse.github.io/Contribute-To-This-Project/ 'https://syknapse.github.io/Contribute-To-This-Project') hissa qoʻshasiz. Unda sizning ismingiz, siz bilan bogʻlanish usullari, qisqa izoh, va oʻzingiz tavsiya qiladigan, web dasturchilar uchun 3ta foydali resurslarga havola aks etadi.

## Hissa qoʻshish

10 bosqichda Ochiq Manbaʼga hissa qoʻshuvchilar safiga qoʻshiling.

### 1-qadam: Loyihani forklang

[Loyiha sahifasining](https://github.com/Syknapse/Contribute-To-This-Project) tepa oʻng tarafdagi  **Fork** tugmasi orqali loyihani forklang. Bu loyihadan oʻzingizning Git Ombori hisobingizga nusxa yaratadi.

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

---

### 2-qadam: Nusxani yuklab oling

Forkni mahalliy qurilmangizga nusxalang:

```bash
git clone https://github.com/sizning-github-foydalanuvchi-nomingiz/Contribute-To-This-Project.git
```

Keyin, loyiha katalogiga oʻting:

```bash
cd Contribute-To-This-Project
```

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

---

### 3-qadam: Yangi novda yarating

Yangi novda yaratib, novdaga oʻting:

```bash
git checkout -b sizning-ismingiz-kartasi
```

Iloji boricha aniq izohlaydigan nom qoʻying — `sizning-ismingiz-kartasi` yaxshi mos tushadi.

**`master` novdada ishlamang.**

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

---

### 4-qadam: Shablon fayldan nusxa oling

Loyihaning karta uchun tayyor shabloni `cards/` jildida mavjud. Undan nusxa oling va nomini Git Omboridagi foydalanuvchi nomingizga mos ravishda oʻzgartiring:

```bash
cp cards/template.html cards/git-omboridagi-foydalanuvchi-nomingiz.html
```

Misol uchun, agar foydalanuvchi nomingiz `octomushuk` boʻlsa:

```bash
cp cards/template.html cards/octomushuk.html
```

**Siz faqatgina shu faylni tahrirlashingiz kerak holos**. `cards/template.html` fayli oʻzgarishsiz qolsin.

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

---

### 5-qadam: Biletni toʻldiring

Yangi yaratgan kartangizni har qanday matn muharriringizda ochib oling:

```bash
code cards/git-ombori-foydalanuvchi-nomingiz.html   # VS Code
# yoki: emacs, nano, vim, etc.
```

Kartangiz shu koʻrinishda boʻladi:

```html
<div class="card">
  <p class="name">Your name</p>
  <p class="contact">
    <!-- Add one or more contact links. At minimum, include your GitHub. -->
    <i class="fab fa-github"></i>
    <a href="https://www.github.com/your_user_handle" target="_blank">Your handle</a>
  </p>
  <p class="about">Write a sentence or two about yourself.</p>
  <div class="resources">
    <p>3 Useful Dev Resources</p>
    <ul>
      <li>
        <a href="#" target="_blank" title="First Resource">Resource 1</a>
      </li>
      <li>
        <a href="#" target="_blank" title="Second Resource">Resource 2</a>
      </li>
      <li>
        <a href="#" target="_blank" title="Third Resource">Resource 3</a>
      </li>
    </ul>
  </div>
</div>
```

Fill it in:

- **Name** —  Yozuvni oʻz ismingiz bilan almashtiring. **`class="name"` oʻzgarishsiz qolsin**.
- **Contact** — Git Ombori havolasi va havola koʻruvchisi uchun foydalanuvchi nomingizni oʻzgartiring. Agar koʻproq havola qoʻshmoqchi boʻlsangiz [Font Awesome Icons](https://fontawesome.com/icons) orqali mos keladigan shrift va namunani izlab topib classʼni almashtiring (misol uchun `fab fa-linkedin`, `fab fa-x-twitter`).
- **About** — Qismni oʻzingizdan kelib chiqib oʻzgartiring. Buni blog postidan koʻra qisqa tvit deb hisoblasangiz boʻladi.
- **Resources** — `#`larni haqiqiy havola bilan almashtiring, `title=""` va havola joyini oʻzgartiring. Bular ihtiyoriy, lekin hohlasangiz ishlaydigan havola kiriting, koʻpi bilan 5ta.

**Eslatma: class nomlarini, hamda, HTML asosini oʻzgartirmang.** PRʼni tekshiruvchi bot siz joʻnatgan PR toʻgʻriligini tekshiruvdan oʻtkizadi.

Tugatib, faylni saqlang.

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

### 6-qadam: Barchasini tekshiring

Karta faylingizni oching va bir boshdan oʻqib chiqing. Ishonch komil qiling: 
- barcha oʻzgartirish kerak boʻlgan maydon toʻgʻirlangan
- havolalar `https://` bilan boshlangan
- fayl nomi Git Ombori foydalanuvchi nomingizga toʻgʻri keladi

Avtomatlashtirilgan jarayon hammasini sinovdan oʻtkazib, asosiy tuzilmadan ortiqcha narsalar boʻlsa oʻtib ketishining oldini oladi.

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

### Step 7: Oʻzgarishlarni topshiring

Kartangiz faylini topshiring:

```bash
git add cards/git-ombori-foydalanuvchi-nomingiz.html
```

Qisqa izoh:

```bash
git commit -m "Mening kartam qoʻshildi"
```

### 9-qadam: PR yuboring (Tortish soʻrovi)

Git Omboridagi forkʼingizga oʻting. U yerda tortish soʻroviga taklif paydo boʻlganini koʻrasiz — **Compare & pull request" tugmasini bosing.

asosiy loyiha `Syknapse/Contribute-To-This-Project` va asosiy novda esa `master` ekanligiga ishonch hosil qiling.

**Allow edits from maintainers** tanlovi belgilanganiga ishonch hosil qilib, **Create pull request** ustiga bosing.

Bot sizning kartangizni avtomatik tarzda koʻrib chiqadi va agar hammasi toʻgʻri boʻlsa qoʻshib yuboradi. Agar bot biror hatolik topsa sizning PR habaringizga izoh qoldiradi, qayerda nima hato qilganingizni — hatoni toʻgʻirlab huddi oʻsha novdaning oʻziga qayta joʻnatsangiz, jarayon qayta takrorlanadi.

[↑ Qoʻllanma boshiga qaytish ↑](#qisqa-kirish-indeksi)

### 10-qadam: Nishonlang

Siz Ochiq Manbaʼga ilk hissangizni qoʼshdingiz! Kartangiz qisqa vaqt ichida [https://syknapse.github.io/Contribute-To-This-Project](https://syknapse.github.io/Contribute-To-This-Project) saytida namoyish qilinadi.

**Mening HTML faylim qayoqga ketdi?** Biq qancha vaqtdan soʻng, joʻnatilgan kartalar loyiha hajmini yengil saqlash uchun avtomatik arxivlanadi. Sizning HTML faylingiz avtomatik tarzda `cards/` jildidan oʻchiriladi, lekin maʼlumoti arxivda tutiladi. Kartangiz web sahifada paydo boʼlsa ham. [Koʻproq bilish uchun](archive/README.md)

## Keyingi qadamlar

- Manabu _tekin_ seriyalar roʻyhatidan qanday qilib hissa qoʻshishni oʻrganishingiz mumkin: [Ochiq Manbaʼli Loyihalarga Hissa Qoʻshish Haqida](https://kcd.im/pull-request)
- Agar siz bu loyihamizni **foydali** deb topgan boʻlsangiz, iltimos sahifa tepasida :star: yulduz :star: qoldiring va loyiha haqida **tvit** ulashish orqali qoʻllab quvvatlang [![Tvit](https://img.shields.io/twitter/url/http/shields.io/twitter/url/http/shields.io.svg?style=social)][tvit]
- Bizning [Discord serverimizga](https://discord.gg/tWkvS4ueVF) qoʻshiling.
- Bu Ochiq Manbaʼli loyiha ekan, karta qoʻshishdan koʻproq hissa qoʻshmoqchi boʻlsangiz [CONTRIBUTING.md](CONTRIBUTING.md) bilan tanishing.
- **Loyihaga hissa qoʻshganingizdan bagʻoyatda mamnunmiz**. Endi siz boshqa loyihalarga hissa qoʻshishda oʻzingizni sinab koʻrsangiz boʻladi; Boshlanishiga osson boʻlishi uchun ![Good First Issue](https://user-images.githubusercontent.com/29199184/33852733-e23b7070-debb-11e7-907b-4e7a03aad436.png) yorligʻi bor masalalarni nishonga oling.
