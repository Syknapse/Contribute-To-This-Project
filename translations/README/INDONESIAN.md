## Pendahuluan

Ini adalah tutorial untuk membantu kontributor pertama kali berpartisipasi dalam proyek yang sederhana dan mudah.

---

### Tujuan

- Melakukan kontribusi ke proyek open source
- Menjadi lebih terbiasa menggunakan GitHub

---

### Untuk siapa ini?

- Untuk pemula absolut. Jika kamu tahu cara menulis dan mengedit tag `<a href="" target=""></a>`, maka kamu bisa melakukannya.
- Juga untuk yang sudah punya sedikit pengalaman tetapi ingin membuat kontribusi open source pertama mereka atau meningkatkan pengalaman dan kepercayaan diri.

---

### Kenapa saya perlu melakukan ini?

Setiap web developer, baik pemula maupun yang sudah berpengalaman, perlu menggunakan Git sebagai version control. GitHub adalah platform hosting Git paling populer dan merupakan pusat komunitas open source.

Menguasai GitHub adalah skill penting. Berkontribusi ke sebuah proyek dapat meningkatkan kepercayaan diri dan memberikan sesuatu yang bisa kamu tampilkan di profil GitHub.

---

### Apa yang akan saya kontribusikan?

![Contributor Card](/readme-only/card.PNG)

Kamu akan menambahkan sebuah kartu ke halaman proyek yang berisi:

- Nama
- Kontak
- Deskripsi singkat
- 3 link resource yang bermanfaat

Kamu hanya perlu menyalin template kartu, mengisinya, lalu mengirimkannya.

---

### Setup

> Catatan: Tutorial ini menggunakan GitHub Desktop

1. Login ke akun GitHub
2. Download GitHub Desktop
3. Alternatif: gunakan terminal atau VS Code

---

### Kontribusi

#### Langkah 1: Fork repository
Salin repository ini ke akun GitHub kamu.

#### Langkah 2: Clone repository
Download repository ke komputer kamu.

#### Langkah 3: Buat branch baru
Gunakan branch untuk memisahkan perubahan dari project utama.

#### Langkah 4: Copy template
- Masuk ke folder `cards/`
- Copy `template.html`
- Rename menjadi `username.html`

#### Langkah 5: Isi kartu
Isi:
- Nama
- Kontak
- Deskripsi
- Resource

#### Langkah 6: Cek hasil
Pastikan:
- Tidak ada placeholder
- Semua link valid

#### Langkah 7: Commit
Simpan perubahan dengan commit.

#### Langkah 8: Push
Upload ke GitHub.

#### Langkah 9: Pull Request
Kirim perubahan ke repository utama.

#### Langkah 10: Selesai 🎉
Selamat! Kamu sudah berkontribusi ke open source.
