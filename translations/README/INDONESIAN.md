## Introduction

Ini adalah tutorial untuk membantu kontributor pertama kali berpartisipasi dalam proyek yang sederhana dan mudah.

### Objectives

* Melakukan kontribusi ke proyek open source.
* Menjadi lebih terbiasa menggunakan GitHub.

### Who is this for?

* Untuk pemula absolut. Jika kamu tahu cara menulis dan mengedit tag `<a href="" target=""></a>`, maka kamu seharusnya bisa melakukannya.
* Juga untuk mereka yang sudah memiliki sedikit pengalaman tetapi ingin membuat kontribusi open source pertama, atau mendapatkan lebih banyak pengalaman dan kepercayaan diri.

### Why do I need to do this?

Setiap web developer, baik yang bercita-cita maupun yang sudah berpengalaman, perlu menggunakan Git sebagai version control, dan GitHub adalah layanan hosting Git paling populer yang digunakan semua orang. Ini juga merupakan pusat komunitas Open Source. Menjadi nyaman menggunakan GitHub adalah keterampilan penting. Berkontribusi ke sebuah proyek meningkatkan kepercayaan diri dan memberi kamu sesuatu untuk ditampilkan di profil GitHub.

Jika kamu adalah developer baru dan bertanya apakah perlu belajar Git dan GitHub, maka jawabannya: ya.

### What am I going to contribute?

![Contributor Card](/readme-only/card.PNG "Contributor Card")

Kamu akan berkontribusi sebuah kartu seperti ini ke [halaman web proyek](https://syknapse.github.io/Contribute-To-This-Project/). Kartu tersebut akan berisi nama kamu, detail kontak, deskripsi singkat, dan 3 link resource berguna untuk web developer yang kamu rekomendasikan.

Kamu akan menyalin file template kartu yang sudah tersedia, mengisinya dengan informasi kamu sendiri, dan mengirimkannya.

> Sudah pernah berkontribusi? Ada cara lain untuk membantu — terjemahan, perbaikan bug, peningkatan aksesibilitas, dan lainnya. Lihat [CONTRIBUTING.md](CONTRIBUTING.md) untuk panduan lengkap.

### Setup! :)

> [!Note]
> Tutorial ini berbasis GitHub Desktop. [Jika kamu nyaman dengan terminal, lihat tutorial ini](/terminal_tutorial.md)

Pertama, mari kita siapkan semuanya.

1. Login ke akun GitHub kamu. Jika belum punya akun, silakan [daftar GitHub](https://github.com/join).
2. Download [GitHub Desktop](https://desktop.github.com/).

   * Alternatif: gunakan terminal.
   * Jika menggunakan VS Code, sudah ada Git bawaan.
   * Cara termudah tetap menggunakan GitHub Desktop.

**Ingin meningkatkan proyek ini?** Terjemahan, bug fix, dan improvement lainnya sangat diterima — tapi buat PR terpisah dari kartu kamu.

### Contribute

Jadilah kontributor open source dalam 10 langkah mudah.

#### Step 1: Fork this repository

* Salin repository ke akun kamu.

| ![Fork](/readme-only/fork.png "click on 'Fork'") |
| :----------------------------------------------: |
|              **Klik tombol *Fork***              |

---

#### Step 2: Clone the repository

* Buat salinan lokal project.

| ![Clone](/readme-only/clone.PNG) |
| :------------------------------: |
| **Klik File → Clone repository** |

---

#### Step 3: Create a new branch

* Buat branch baru agar perubahan terpisah dari `master`.

---

#### Step 4: Copy the template file

* Masuk folder `cards/`
* Copy `template.html`
* Rename jadi `username.html`

---

#### Step 5: Fill in your card

```html
<div class="card">
  <p class="name">Nama kamu</p>
</div>
```

---

#### Step 6: Check your work

* Pastikan tidak ada error

---

#### Step 7: Commit your changes

* Simpan perubahan

---

#### Step 8: Push your changes to GitHub

* Upload ke GitHub

---

#### Step 9: Submit a PR

* Klik "Compare & pull request"

---

#### Step 10: Celebrate 🎉

Selamat! Kamu berhasil kontribusi.
