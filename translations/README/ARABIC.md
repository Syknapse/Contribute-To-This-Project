# ساهم في هذا المشروع

هذا المشروع عبارة عن دليل عملي مبسط يساعد المبتدئين على تقديم أول مساهمة لهم في مشاريع المصدر المفتوح باستخدام GitHub.

---

## 📌 مقدمة

إذا كنت جديدًا في GitHub أو لم تقم بالمساهمة في مشروع مفتوح المصدر من قبل، فهذا المشروع مناسب لك تمامًا.
سيساعدك هذا الدليل خطوة بخطوة على فهم كيفية العمل مع GitHub وتقديم أول Pull Request لك.

---

## 🎯 الهدف

* تعلم كيفية المساهمة في مشاريع مفتوحة المصدر
* فهم أساسيات Git و GitHub
* تنفيذ أول Pull Request بنجاح

---

## 👥 لمن هذا المشروع؟

هذا المشروع مخصص لـ:

* المبتدئين في GitHub
* الطلاب والمطورين الجدد
* أي شخص يريد البدء في المصدر المفتوح

---

## 🚀 كيف تساهم؟

اتبع الخطوات التالية بعناية:

### 1. قم بعمل Fork للمستودع

اضغط على زر **Fork** لإنشاء نسخة من المشروع على حسابك.

---

### 2. انسخ المشروع (Clone)

قم بتنزيل المشروع على جهازك باستخدام Git:

```bash
git clone https://github.com/YOUR-USERNAME/Contribute-To-This-Project.git
```

---

### 3. أنشئ فرعًا جديدًا (Branch)

```bash
git checkout -b my-contribution
```

---

### 4. قم بإجراء التعديلات

* افتح المشروع في محرر الأكواد
* أضف التعديل المطلوب (مثل إضافة اسمك أو تحديث ملف)

---

### 5. احفظ التغييرات (Commit)

```bash
git add .
git commit -m "Add my contribution"
```

---

### 6. ارفع التغييرات (Push)

```bash
git push origin my-contribution
```

---

### 7. أنشئ Pull Request

* اذهب إلى GitHub
* اضغط على **Compare & Pull Request**
* أرسل طلبك

---

## ⏱️ الوقت المتوقع

يمكنك إكمال هذه العملية في أقل من 30 دقيقة.

---

## 💡 نصائح مهمة

* تأكد من قراءة التعليمات جيدًا
* لا تستخدم HTML (فقط Markdown)
* تأكد من أن التعديلات واضحة وصحيحة
* التزم بنفس تنسيق الملف الأصلي

---

## 🎉 بعد المساهمة

* سيتم مراجعة Pull Request الخاص بك
* قد يُطلب منك تعديل بعض الأمور
* بعد القبول، سيتم دمج مساهمتك في المشروع

---

## 🌟 الخطوة التالية

* جرّب المساهمة في مشاريع أخرى
* طوّر مهاراتك في Git و GitHub
* شارك هذا المشروع مع الآخرين
