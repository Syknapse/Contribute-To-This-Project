## Soʻz boshi

Qoʻllanma birinchi bor contribute qilmoqchi boʻlganlarga kichik va sodda loyihada qatnashish orqali malaka olishda yordam beradi.

### Maqsadlar

- Ochiq manbaʼli loyihalarga hissa qoʻshish.
- Git Omborini ishlatishda koʻnikma hosil qilish.

### Bu kim uchun?

- Yangi boshlovchilar. Agar langar teg `<a href="" target=""></a>` yozishni bilsangiz, qoʻlingizdan keladi.
- Ozgina tajribali lekin ochiq manbaʼga hech hissa qoʻshib koʻrmaganlar, yoki bilimini oʻtkirlamoqchi boʻlgan har kim. 

### Nega buni bajarishingiz lozim?

Har qanday dasturchi, kirib kelayotgan yoki tajribali, Git talqin boshqaruvini ishlatishi kerak, va Git Ombori hamma ishlatadigan eng mashhur Git xosting servisi hisoblanadi. Va u Ochiq Manba jamiyatining yuragi ham. Git Omboridan foydalanish mahorati bu juda ham kerakli hunar. Loyihaga hissa qoʻshish oʻzingizga ishonchni orttirishga va Git Omboridagi hisobingizda nimadir koʻrsatish uchun hizmat qiladi.

Agar siz dasturlashda yangi boʻlsangiz, hamda, Git va Git Omborini oʻrganaymikin yoki yoʻq deb oʻylanayotgan boʻlsangiz bu siz uchun javob boʻla oladi: [Siz Gitʼni kecha oʻrganishingiz kerak edi](https://codeburst.io/number-one-piece-of-advice-for-new-developers-ddd08abc8bfa 'Yangi Dasturchi? Siz Gitʼni kecha oʻrganishingiz kerak edi. Brendon Morelli, CodeBurst.io yaratuvchisi').

### Qanday hissa qoʻshishingiz mumkin?

![Hissador Kartasi](/readme-only/card.PNG 'Hissador Kartasi')

Siz huddi manashunday karta toʻldirish orqali [loyiha web sahifasiga](https://syknapse.github.io/Contribute-To-This-Project/ 'https://syknapse.github.io/Contribute-To-This-Project') hissa qoʻshasiz. Unda sizning ismingiz, siz bilan bogʻlanish usullari, qisqa izoh, va oʻzingiz tavsiya qiladigan, web dasturchilar uchun 3ta foydali resurslarga havola aks etadi.

Bunda siz tayyor shablon fayldan nusxa olib, oʻz maʼlumotlaringiz bilan toʻldiringu topshiring.

#### 3-qadam: Yangi novda yarating

- Repo yuklab boʻlindi ham deylik, endi yangi boʻlim yaratish vaqti keldi.

| :---------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------: |
| :arrow_right_hook: **Novdaga nom bering, hamda `Create branch` ustiga bosing** |                           ![Novdani nomlang](/readme-only/branch-name.PNG 'Name your branch')                            |

#### 4-qadam: Shablon fayldan nusxa oling

- Loyihaning karta uchun tayyor shabloni `cards/` jildida mavjud.
- Loyiha jildini kompyuteringizdan izlab toping. Agar standart manzilni oʻzgartirmagan boʻlsangiz `Documents > GitHub > Contribute-To-This-Project > cards` manzilidan topishingiz mumkin.
- `cards/` jildida `template.html` faylini koʻrasiz. Anashu sizning boshlangʻich nuqtangiz.
- `template.html`dan nusxa oling va nuxsa nomini `sizning-git-omboridagi-foydalanuvchi-nomingiz.html`ga oʻzgartiring. Misol uchun, agar Git Omboridagi foydalanuvchi nomingiz `octomushuk` boʻlsa, sizning faylingiz `octomushuk.html` deb nomlanishi kerak.

Agar siz terminaldan foydalanayotgan boʻlsangiz:

```bash
cp cards/template.html cards/sizning-git-omboridagi-foydalanuvchi-nomingiz.html
```

- Endi `cards/sizning-git-omboridagi-foydalanuvchi-nomingiz.html`ni kod tahrirlovchingizda oching. Siz faqatgina shu faylni tahrirlashingiz kerak boʻladi.

#### 5-qadam: Biletni toʻldiring

- Sizning kartangiz manashunday koʻrinishda boʻladi:

```html
<div class="card">
  <p class="name">Ismingiz</p>
  <p class="contact">
    <!-- Bogʻlanish uchun bir yoki bir nechta havolalarni kiriting. Kamida Git Omboringizni. -->
    <i class="fab fa-github"></i>
    <a href="https://www.github.com/foydalanuvchi_nomingiz" target="_blank">Foydalanuvchi nomingiz</a>
  </p>
  <p class="about">Oʻzingiz haqingizda bir-ikki ogʻiz soʻz</p>
  <div class="resources">
    <p>3 Useful Dev Resources</p>
    <ul>
      <li>
        <a href="#" target="_blank" title="Birinchi Resurs">1-resurs</a>
      </li>
      <li>
        <a href="#" target="_blank" title="Ikkinchi Resurs">2-resurs</a>
      </li>
      <li>
        <a href="#" target="_blank" title="Uchinchi Resurs">3-resurs</a>
      </li>
    </ul>
  </div>
</div>
```

- `Ismingiz` yozuvini oʻz ismingiz bilan almashtiring. **`class="name"` oʻzgarishsiz qolsin**.
- Git Ombori havolasi va havola koʻruvchisi uchun foydalanuvchi nomingizni oʻzgartiring. Agar koʻproq havola qoʻshmoqchi boʻlsangiz [Font Awesome Icons](https://fontawesome.com/icons) orqali mos keladigan shrift va namunani izlab topib classʼni almashtiring (misol uchun `fab fa-linkedin`, `fab fa-x-twitter`).
- `Oʻzingiz haqingizda bir-ikki ogʻiz soʻz` qismini ham oʻzingizdan kelib chiqib oʻzgartiring. Buni blog postidan koʻra qisqa tvit deb hisoblasangiz boʻladi.
- Har bir resurs uchun `#`ni haqiqiy havola bilan almashtiring, `title=""` qismiga qisqa izoh qoʻshing. Web dasturchilar uchun har qanday foydali deb biladigan video, nutq, podkast, maqola, ish qurolingizni yozib qoldiring. Agarda siz endigina boshlagan boʻlsangiz bundan hecham oʻylanmang, har qanday biladigan narsangizni ulashsangiz boʻladi, hatto uni sodda deb hisoblasangiz ham. Qancha kishiga bu foydali kelishi sizni ajablantirishi turgan hol.
  - **Havola** `href="https://biror-havola.com"` — faqatgina toʻliq havolalar, qisqartmalardan foydalanmang.
  - **Sarlavha** `title="Qisqa izoh"` — bir nechta soʻz yetadi.
  - **Nom** Yozuv `>Resurs nomi</a>`
  
- **hammasini saqlaganingizga** ishonch komil qiling.

- **Eslatma: class nomlarini, hamda, HTML asosini oʻzgartirmang.** PRʼni tekshiruvchi bot siz joʻnatgan PR toʻgʻriligini tekshiruvdan oʻtkizadi.

#### 9-qadam: PR yuboring (Tortish soʻrovi)

- Siz kutgan vaqt keldi; _Tortish soʻrovi_ (Pull Request = PR) yuborish.
- Shu paytgacha barcha qilgan ishlaringiz loyihaning alohida novdasida edi, esingizda boʻlganidek Git Omboridagi hisobingizda joylashgan.
- Endi qilgan oʻzgarishlaringizni asosiy loyihaga (novdaga) joʻnatib, birlashtirish vaqti keldi.
- Bu [ Tortish Soʻrovi ](https://help.github.com/articles/about-pull-requests/ 'Tortish Soʻrovlari haqida - GitHub Yordam') deb ataladi, chunki siz haqiqiy loyihaning boshqaruvchisidan oʻzgarishlaringizni asosiy loyihaga "tortishni" soʻramoqdasiz.
- **Sizning novdangizning** Git Omboridagi asosiy sarlavhasiga boring (u yerda fork belgisi va oʻz foydalanuvchi nomingizni tepa qismida koʻrishingiz mumkin)
- Tepa tomonda soʻrov habarini aks etganini yashil tugma bilan birga koʻrishingiz mumkin.

|  ![PR oching](/readme-only/pull-request.PNG 'Bu odatda sahifaning tepa tomonida aks etadi, izoh ostida, loyiha jild va fayllari ustida')  |
| :--------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
|                                                    :arrow_right_hook: **`Compare and pull request` ustiga bosing**                                                     |
| ![PR oching](/readme-only/pull-request-branches.PNG 'Siz oʻzingizning forkʼdagi novdadan haqiqiy loyihadagi asosiy master novdaga tortib, birlashtirishni soʻramoqdasiz') |
|                                               :arrow_right_hook: Bu esa `Open a pull request` sahifasining koʻrinishi.                                               |

- ESLATMA: _siz oʻz novdangizni haqiqiy loyiha bilan birlashtirishga urinmoqdasiz, oʻz forkʼingizdagi `Master` novda bilan emas_.
- Pastdagi ushbu suratdan siz joʻnatgan Tortish Soʻrovi qanday koʻrinishda boʻlishi kerakligini koʻrsangiz boʻladi.

- Koʻp tugmalarni koʻrib havotirlanmang. Hozircha shu 3 bosqichni qilsangiz yetadi.
- `Allow edits from maintainers`ni belgilash esingizdan chiqmasin.
- Mana endi _Tortish Soʻrovi_ loyihaga joʻnatildi. Bot avtomatik ravishda kartangizni tekshirib, agar barchasi yaxshi boʻlsa, loyiha bilan birlashtirib yuboradi. Barchasi tugaganidan soʻng Git Omboridan bildirishnoma olasiz.

### Keyingi Qadamlar

- Bir qancha vaqtdan soʻng Tortish Soʻrovingiz qoʻshilganini tekshirish uchun qayting.

- Bizning [Discord serverimizga](https://discord.gg/tWkvS4ueVF) qoʻshilishingiz mumkin.
- Menga [𝕏 (Twitterda)](https://twitter.com/Syknapse '@Syknapse') yoki [bu yerda](https://syknapse.github.io/Syk-Houdeib/#contact 'Men bilan bogʻlanish usullari | Portfolio') berilgan usullar orqali **obuna** boʻling.
- Bu Ochiq Manbaʼli loyiha ekan, karta qoʻshishdan tashqari muammolarni toʻgʻirlash, tuzatishlar, yoki yangi xususiyatlar qoʻshish orqali ham hissa qoʻshishingiz mumkin. [Masala](https://help.github.com/articles/creating-an-issue/ 'Muammolarga Ustalik qilish | GitHub Qoʻllanmalari') oching yoki yangi [tortish soʻrovi](https://help.github.com/articles/creating-a-pull-request-from-a-fork/ 'Forkʼdan yangi Tortish Soʻrovi yaratish') joʻnating.
